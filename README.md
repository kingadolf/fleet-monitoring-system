🚛 Fleet Monitoring & Petty Cash Management System

A full-stack web application designed to monitor vehicle fleet operations and manage petty cash transactions in real-time. Built for operational efficiency, tracking, and reporting.

🔥 Features
  🚚 Truck / Vehicle tracking system (status, fuel level, driver info)  
  ⛽ Fuel monitoring with visual analytics
  📊 Interactive dashboards (charts & KPIs)
  💰 Petty cash management system
  📁 Transaction history & reporting
  🔍 Search, filter, and CRUD operations
  📱 Responsive UI for desktop and mobile
🛠 Tech Stack
  Frontend: React.js, Recharts, CSS
  Backend: Node.js, Express.js
  Database: MongoDB
  Deployment: (Vercel / Render ready)
📈 Purpose

  This system helps companies track fleet status, monitor fuel usage, and manage operational expenses efficiently in one centralized dashboard.
